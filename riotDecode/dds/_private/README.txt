To enable dds image viewing you need to install on your computer:

1 - DDS Utilities from NVIDIA

Info : https://developer.nvidia.com/legacy-texture-tools

2 - ImageMagick

Info : http://www.imagemagick.org

3 - Edit riotDecode/dds/DdsFile.php
provide path to readdxt.exe (NVIDIA DDS Utilities)
provide path to convert.exe (ImageMagick)
